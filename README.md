# covert_gbk.pl
This script unifies extractUpStreamDNA.pl, gbk2faaAV.pl, gbk2ffn.pl. and GBKtoPTT.pl which allows the user to take a GenBank flatfile and extract DNA upstream from each gene, and  convert gbk file into Protein Table file (*.ptt), fasta-formatted proteins (*.faa) and fasta-formatted CDS sequences (*.ffn).
